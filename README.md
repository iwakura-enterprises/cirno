# Cirno

<div align="center">
  <img src="https://akasha.iwakura.enterprises/data-source/hetzner/public/logo/cirno.png" alt="Cirno logo" width="300" border-effect="rounded"/>
</div>

Cirno is a Java library serving common functionalities when creating Java applications. It offers
utilities when working with collections, lists, numbers, exceptions, completable futures but also
efficient string operations (Boyer Moore's string search, Jaro-Winkler similarity).
